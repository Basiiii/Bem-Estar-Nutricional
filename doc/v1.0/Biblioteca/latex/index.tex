 A Biblioteca {\ttfamily Gestor\+Dados\+IO} é responsável por fornecer funcionalidades de importação e exportação de dados a partir de ficheiros. Esta biblioteca é parte integrante do projeto Bem Estar, que tem como objetivo apoiar o bem-\/estar e cuidados nutricionais.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Funcionalidades}{Funcionalidades}}\label{index_autotoc_md1}
\hypertarget{index_autotoc_md2}{}\doxysubsection{\texorpdfstring{Importar Dados}{Importar Dados}}\label{index_autotoc_md2}
A biblioteca suporta a importação de diferentes tipos de dados, incluindo pacientes, dietas e planos nutricionais. Para importar dados, utilize as seguintes funções\+:


\begin{DoxyItemize}
\item {\ttfamily Importar\+Dados\+Fich\+Binario}\+: Importa dados de um ficheiro binário.
\item {\ttfamily Importar\+Pacientes\+Fich\+Texto}\+: Importa dados de pacientes de um ficheiro de texto.
\item {\ttfamily Importar\+Dietas\+Fich\+Texto}\+: Importa dados de dietas de um ficheiro de texto.
\item {\ttfamily Importar\+Planos\+Fich\+Texto}\+: Importa dados de planos de um ficheiro de texto.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysubsection{\texorpdfstring{Exportar Dados}{Exportar Dados}}\label{index_autotoc_md3}
Além da importação, a biblioteca permite exportar dados para ficheiros. A função correspondente é\+:


\begin{DoxyItemize}
\item {\ttfamily Exportar\+Dados\+Fich\+Binario}\+: Exporta dados para um ficheiro binário. 
\end{DoxyItemize}
